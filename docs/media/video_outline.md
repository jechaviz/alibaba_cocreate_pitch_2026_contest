# Demo Video Outline

Target length: 90 seconds.

Audience: Alibaba CoCreate Pitch / Accio Work judges.

## Structure

| Time | Visual | Voiceover |
|---:|---|---|
| 0-7s | Open demo cockpit with supplier, economics, and workflow visible | "Sourcing is no longer the hard part. The hard part is knowing whether a supplier quote deserves your cash." |
| 7-18s | Compare supplier shortlist | "Search-only tools give options. Accio Commerce Copilot turns options into a decision graph." |
| 18-34s | Change price, CAC, and freight | "The model updates landed cost, contribution margin, cash required, break-even units, and payback in real time." |
| 34-50s | Switch supplier and show risk memo | "A cheaper supplier can still lose if lead time, defect risk, or compliance gaps break the launch." |
| 50-64s | Run Accio action | "Accio Work becomes the execution layer: brief, RFQ, risk memo, storefront copy, and approval handoff." |
| 64-78s | Show competitive edge / judge scorecard | "Unlike spreadsheets or generic agents, every recommendation stays tied to evidence and next actions." |
| 78-90s | Submission packet/evidence export | "Commerce AI should not stop at search. It should help small businesses buy with confidence." |

## Capture Checklist

- Desktop viewport at 1440x900.
- Mobile viewport proof at 390x844 for responsive layout.
- Show one complete scenario, not every feature.
- Keep narration product-focused and avoid implementation jargon.
- End with product name, track, and repository/demo links.
- Say "decision cockpit" at least once and "supplier search is not enough" once.

## Required Assets

- Demo UI running locally or deployed.
- V CLI evidence JSON exported.
- Submission packet open beside the UI for final proof.
- Optional Accio Work page/form screenshot from an authorized session.
