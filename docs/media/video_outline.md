# Demo Video Outline

Target length: 90 seconds.

Audience: Alibaba CoCreate Pitch / Accio Work judges.

## Structure

| Time | Visual | Voiceover |
|---:|---|---|
| 0-8s | Open demo cockpit with category selected | "Small importers can find products, but deciding what is actually worth buying is still risky." |
| 8-20s | Supplier comparison table | "Accio Commerce Copilot imports supplier options and normalizes MOQ, price, freight, lead time, and reliability signals." |
| 20-38s | Unit economics panel updates | "The AI turns quotes into landed cost, contribution margin, cash required, break-even units, and payback." |
| 38-55s | Risk dashboard and mitigation tasks | "It explains supplier, compliance, and cash risks, then creates the next actions before a sample order." |
| 55-72s | Recommended buy plan and export | "The founder gets a buy/no-buy recommendation, negotiation script, and launch plan." |
| 72-90s | Submission packet/evidence export | "Commerce AI should not stop at search. It should help small businesses buy with confidence." |

## Capture Checklist

- Desktop viewport at 1440x900.
- Mobile viewport proof at 390x844 for responsive layout.
- Show one complete scenario, not every feature.
- Keep narration product-focused and avoid implementation jargon.
- End with product name, track, and repository/demo links.

## Required Assets

- Demo UI running locally or deployed.
- V CLI evidence JSON exported.
- Submission packet open beside the UI for final proof.
- Optional Accio Work page/form screenshot from an authorized session.
